# Schnuffelll Bot - Panel Manager

## Quick Start

1. Upload ZIP ke panel Pterodactyl
2. Edit config.js dengan token bot dan owner ID
3. npm install
4. npm start

## Version: 6.0 (27 Desember 2025)

## Features
- Panel Management V1-V5
- VPS DigitalOcean Manager  
- CPU Monitoring Auto-Stop
- Live Auto-Update

## Update: /update command available.
